
    1. Обслуживающая система.
    Эта группа содержит один выпадающий список:
       1. Число машин.
		O: произвольное число машин
		O1: одна-единственная машина
		O2: 2 машины
		O3: 3 машины
		Om: фиксированное число машин

    2. Характеристики работ.
    Четыре выпадающих списка.
       1. Прерывания.
		: прерывания запрещены
		pmtn: прерывания разрешены
       2. Время обслуживания.
		: произвольные длительности обслуживания
		p_ij=1: единичные длительности обслуживания
       3. Моменты поступления.
		: моменты поступления всех работ равны нулю
		r_j: заданы произвольные моменты поступления работ
       4. Отношеня предшествования.
		: не заданы отношения предшествования
		prec: произвольный ацикличный ориентированный граф
		tree: дерево
		in-tree: входящее дерево
		out-tree: выходящее дерево
		chains: набор цепей

    3. Целевая функция.
    Один выпадающий список (с таким же названием)
		C_max: общий момент завершения обслуживания C_max = max{C_j}
		L_max: максимальное запаздывание L_max = max{L_j}
		sum(C_j): сумма моментов завершения
		sum(U_j): число запаздывающих работ
		sum(T_j): суммарное запаздывание
		sum(w_j*C_j): взвешенная сумма моментов завершения
		sum(w_j*U_j): взвешенное число запаздывающих работ
		sum(w_j*T_j): взвешенное суммарное запаздывание

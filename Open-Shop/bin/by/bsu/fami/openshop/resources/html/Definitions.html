Обозначения

 

Условие задач построения оптимального расписания принято кодировать в формате , состоящем из трёх полей. Рассмотрим в отдельности каждое из полей:

1.    – описание обслуживающей среды.

, где обозначает пустой символ. Первые четыре символа служат для обозначения одностадийной системы обслуживания, остальные – для многостадийной.

·         : одна машина; ;

·         : параллельные одинаковые машины; ;

·         : параллельные машины разной производительности; , где – производительность машины;

·         : параллельные различные машины; – произвольные целые неотрицательные числа;

·         : Open Shop; – произвольные целые неотрицательные числа;

·         : Job Shop; – произвольные целые неотрицательные числа;

·         : Flow Shop; – произвольные целые неотрицательные числа;

– число машин. Запись подразумевает фиксированное число , означает, что число приборов – переменная величина, определяемая конкретным входом задачи.

2.    описывает характеристики работ. определены следующим образом:

·         : прерывания обслуживания любой работы запрещены;

·         : прерывания разрешены;

·         : длительности времени обслуживания либо произвольны;

·         : одинаковы и равны для всех работ и машин;

·         : одинаковы и равны для всех работ и машин;

·         : моменты поступления всех работ равны нулю;

·         : заданы произвольные моменты поступления работ ;

·         : на множестве работ не заданы отношения предшествования;

·         : граф отношений предшествования суть: произвольный ацикличный ориентированный граф, дерево, входящее дерево, выходящее дерево, последовательно-параллельный, набором цепей соответственно;

3.    описывает критерий задачи, который состоит в отыскании расписания, допустимого относительно директивных сроков либо минимизирующих некоторый целевой функционал. Значение целевого функционала определяется следующими величинами, вычисляемыми для каждой работы при заданном расписании:

·         момент завершения обслуживания ;

·         временное смещение ;

·         запаздывание ;

·         единичный штраф , где , если работа является запаздывающей (), иначе ;

Рассматриваются следующие целевые функции:

·         максимальная стоимость ;

·         общий момент завершения обслуживания ;

·         максимальное временное смещение ;

·         максимальное запаздывание ;

·         суммарная стоимость ;

·         сумма моментов завершения ;

·         число запаздывающих работ ;

·         суммарное запаздывание ;

·         взвешенная сумма моментов завершения ;

·         взвешенное число запаздывающих работ ;

·         взвешенное суммарное запаздывание ;

То есть .

Нахождение эффективно нормальных классов с использованием нестрогого суммирования векторов

Определение Допустимое расписание длины Lmax называется нормальным. Пример задачи open shop, обладающий нормальным расписанием, называется нормальным. Подмножество нормальных примеров называется нормальным классом. Если существует алгоритм полиномиальной трудоемкости, который для любого примера из заданного нормального класса находит его нормальное (а следовательно, оптимальное) расписание, то нормальный класс называется эффективно-нормальным классом.

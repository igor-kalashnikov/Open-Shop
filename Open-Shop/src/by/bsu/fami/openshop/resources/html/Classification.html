Введение

Лагевег и др. в 1981-1982 разработали MSPCLASS - компьютерную программу для автоматической классификации задач основанных на теории расписаний. На основание альфа| бетта| гамма -схемы классификации аль Грэм и др..(1979) MSPCLASS расчитывает проблемы, которые: - максимально полиномиально разрешимы: самые сложные задачи, которые решаются за полиномиальное время, т.е. задачи, для которых известны полиномиальные решения, но для частных - сложных - случаев не разрешимы полиномиально -максимально псевдополиномиально разрешимы: самые сложные задачи, которые псевдополиномиально(но не полимиально) разрешимы -минимально NP-сложные: простые задачи, которые являются NP-сложными, то есть задачи, которые известны как NP-сложные, но простейшие случае не являются NP-сложными -минимально открытые: задачи, для которых сложность решения не известна, но для всех простейших случаев решения полиномиальные -максимально открытые: задачи, для которых сложность решения неизвестна, но в в худшем случае решения NP-сложные

На вход MSPCLASS подаются задачи, известные как полиномиально разрешимые, или NP-сложные, и элементарные правила для построения расписания.

  • Расширение классификации на новые классы задач основанных на теории расписаний.
